
# Fish configuration

## Files

- **Main configuration file**: `config.fish` -> `~/.config/fish/config.fish`
- **Variables script**: `variables.fish` -> `~/.config/fish/variables.fish`
- **Local script**: `~/.config/fish/local.fish`
- **Aliases script**: `aliases.fish` -> `~/.config/fish/aliases.fish`
- **Prompt script**: `functions/fish_prompt.fish` -> `~/.config/fish/functions/fish_prompt.fish`
- **Greeting script**: `functions/fish_greeting.fish` -> `~/.config/fish/functions/fish_greeting.fish`

> [!NOTE]
> The prompt script is natively handled by fish itself, the version in this repository is currently just the default.
